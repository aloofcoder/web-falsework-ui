### 项目简介
[web-falsework-ui](https://github.com/aloofcoder/web-falsework-ui) 是一个通用后台管理系统，基于vue + element ui + vue admin template 等实现，该项目是[web-falsework](https://github.com/aloofcoder/web-falsework)的前端项目 。
